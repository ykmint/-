# 串口调试助手开发项目  

## 项目简介  

本项目为串口调试助手的开发，旨在提供一个简洁易用的工具，帮助用户进行串口通信的调试和监控。该项目使用 Python 和 PyQt5 进行开发，用户界面由 Qt Designer 设计。  

## 技术栈  

- **编程语言**: Python  
- **图形用户界面**: PyQt5  
- **界面设计工具**: Qt Designer
